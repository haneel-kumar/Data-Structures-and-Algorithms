Algorithm Steps
  • 1
    Build charFrequency map from the pattern (not the string). Count how many times each distinct character appears.
  • 2
    Slide a fixed window of exactly pattern.length across the string. For each new char entering: if it's in the map, decrement its count. If count hits 0, increment isMatch.
  • 3
    If isMatch === Object.keys(charFrequency).length, every pattern char is fully satisfied → return true.
  • 4
    When window exceeds pattern length, slide by removing the leftmost char. If that char is in the map: if its count was 0 (was fully matched), decrement isMatch first; then increment its count back.
O(N + M) time — N = string, M = pattern
O(M) space — at most M distinct chars in map
